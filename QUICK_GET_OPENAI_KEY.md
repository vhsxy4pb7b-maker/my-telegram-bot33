# 🔑 快速获取 OpenAI API 密钥

## 🚀 3步获取密钥

### 步骤 1: 访问 OpenAI

打开: https://platform.openai.com/api-keys

### 步骤 2: 创建密钥

1. 登录 OpenAI 账号（如果没有，先注册）
2. 点击 **"Create new secret key"**
3. **立即复制密钥**（格式: `sk-...` 或 `sk-proj-...`）

### 步骤 3: 添加到 Zeabur

1. 在 Zeabur 项目页面 → 应用服务 → Environment Variables
2. 找到或添加 `OPENAI_API_KEY`
3. 粘贴刚才复制的密钥
4. 保存

---

## ⚠️ 重要提示

- 密钥**只会显示一次**，立即复制保存
- 格式应该以 `sk-` 开头
- 不要在代码中硬编码密钥

---

## 💰 费用

- 新账号通常有 $5 免费额度
- 超出后按使用量付费
- 查看使用: https://platform.openai.com/usage
