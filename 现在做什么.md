## 📋 接下来3个简单步骤

### 步骤 1: 在 Railway 创建项目（2分钟）

1. **打开浏览器**，访问: https://railway.app
2. 点击页面上的 **"New Project"** 按钮（通常在右上角）
3. 选择 **"Deploy from GitHub repo"**
4. 如果提示授权，点击 **"Authorize Railway"** 授权 GitHub
5. 在仓库列表中找到并点击: **`my-telegram-bot33`**
6. 等待部署开始（会看到构建进度）

**完成后告诉我**: "项目已创建" 或 "部署中"

---

### 步骤 2: 添加数据库（1分钟）

1. 在项目页面，点击 **"New"** 按钮
2. 选择 **"Database"**
3. 选择 **"Add PostgreSQL"**
4. 完成！Railway 会自动设置数据库

**完成后告诉我**: "数据库已添加"

---

### 步骤 3: 配置环境变量（5分钟）

1. 在项目页面，点击你的**应用服务**（不是数据库）
2. 点击 **"Variables"** 标签
3. 点击 **"New Variable"** 按钮
4. 添加以下8个变量（一个一个添加）：

```
变量名: FACEBOOK_APP_ID
值: 你的Facebook应用ID

变量名: FACEBOOK_APP_SECRET  
值: 你的Facebook应用密钥

变量名: FACEBOOK_ACCESS_TOKEN
值: 你的Facebook访问令牌

变量名: FACEBOOK_VERIFY_TOKEN
值: 任意字符串（比如: my_verify_token_123）

变量名: OPENAI_API_KEY
值: 你的OpenAI API密钥

变量名: TELEGRAM_BOT_TOKEN
值: 你的Telegram Bot令牌

变量名: TELEGRAM_CHAT_ID
值: 你的Telegram聊天ID

变量名: SECRET_KEY
值: YB-Y7XHm6JuFqMl1fJOuFRLgUEJZPG2x5lQnVC_tJ2U
```

**完成后告诉我**: "环境变量已配置"
